Tokenizing the description Column

Well need to split the description column across each whitespace character found in our descriptions. From there, well need to add the priceRange category into our data frame of word tokens. Perhaps not surprising, the most frequent words are low-information words. They dont really tell us about the properties these words/tokens are commonly referred to as stopwords. Ill read in a dictionary of common stopwords and then we can use what we know about Python merges (specifically anti-joins) in order to filter out the stopwords.

Removing Stopwords

As mentioned, stopwords are commonly occurring tokens that are unlikely to provide useful information for our analyses. There are several lists of stopwords available I found one that I like here. We could even add to this list if we wanted (and we probably do, since we know that all of these descriptions have to do with homes). Ill leave that to you as well though.

Lets read in the data frame of stopwords.

stopwords = pd.read_csv("https://raw.githubusercontent.com/juliasilge/tidytext/main/data-raw/stop_words.csv")

We can see the first few stopwords below, ignoring the lexicon column.

Now lets use an anti-join to remove those words!

temp_list_outer = tokenized_descriptions.merge(stopwords, how = "outer", indicator = True, left_on = "word", right_on = "word")

tokenized_no_stopwords = temp_list_outer[(temp_list_outer._merge == "left_only")].drop(["_merge"], axis = 1).reset_index()

That looks better! There are still some issues here with spaces, symbols, and special characters. We can remove those with regular expressions. Ill leave most of this to you, but below Ill do a bit of naive filtering to remove any tokens containing fewer than 5 characters and also any NULL tokens.

tokenized_no_stopwords = tokenized_no_stopwords[tokenized_no_stopwords["word"].notnull()]
tokenized_no_stopwords = tokenized_no_stopwords[tokenized_no_stopwords["word"].apply(lambda x: len(x) > 4)]

A Plot of Most Frequent Words in Descriptions

Lets finish this off by producing a barplot of the most commonly occurring words in each priceRange category.

tokenized_description_counts = tokenized_no_stopwords.groupby(["priceRange", "word"]).size().reset_index()
tokenized_description_counts.columns = ["priceRange", "word", "count"]
tokenized_description_counts = tokenized_description_counts.sort_values("count", ascending = False)

(
  ggplot(data = tokenized_description_counts[tokenized_description_counts.groupby('priceRange')['count'].rank(ascending=False) <= 15]) +
  geom_col(aes(x = "word", y = "count", fill = "priceRange"), show_legend = False) + 
  coord_flip() +
  facet_wrap("~priceRange", scales = "free")
)
